# order_by_birthday

# 追加予定機能

- 誕生日を和暦で表示
- 年号毎(昭和、平成、令和)が誕生日のユーザーを抽出
- 年齢を表示

- 指定した年(和暦でも西暦でも)の誕生日が祝日であるかを表示
- 和暦で誕生日を登録できるようにする
- 誕生日の干支を表示
- 元号毎の天皇陛下をフルネームを表示する
- 誕生日年を入れた時に元号がどれくらい続いたかの年数を表示
- 和暦が何時代（江戸時代、安土桃山、平安、鎌倉）なのかを表示
